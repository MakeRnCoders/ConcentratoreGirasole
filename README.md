Concentratore Girasole
================

Motorized solar concentrator with tracker for Arduino

The sensor is made of 4 photoresistors and Arduino uses 3 relays to drive 2 motors:
* 3 Relays 12V AC 8A
* 3 Transistors NPN BD237
* 3 Resistors 10K Ohm
* 3 Diodes
